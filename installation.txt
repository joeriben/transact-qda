transact-qda — Installation


Voraussetzungen
---------------

  Docker Engine 24+ oder Docker Desktop
  Git


Installieren und starten
------------------------

  1. Repository klonen:
         git clone https://github.com/joeriben/transact-qda.git
         cd transact-qda

  2. Session-Secret setzen:
         echo "SESSION_SECRET=$(openssl rand -hex 32)" > .env

  3. Starten:
         docker compose up -d

     Der erste Start baut das Image (ca. 5 Minuten). Der erste
     KI-Aufruf lädt das Embedding-Modell nomic-embed-text-v1.5
     (~150 MB). Beides ist einmalig.

Für einen anderen Host-Port als 5174: TQDA_PORT=... in .env setzen.


Erster Start
------------

http://localhost:5174 im Browser öffnen.
Mit Benutzername admin und Passwort adminadmin anmelden.
Passwort am gelben Banner ändern.


Alltägliche Nutzung
-------------------

    docker compose up -d       starten
    docker compose down        stoppen (Daten bleiben in Volumes)
    docker compose logs -f     Logs anzeigen


Aktualisieren
-------------

    git pull
    docker compose build
    docker compose up -d
